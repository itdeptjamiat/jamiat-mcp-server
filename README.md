# jamiat-mcp-server

A simple MCP server for managing Jamiat's IT projects.

## Setup

1. Install dependencies:

   ```bash
   uv sync
   ```

2. Run the server:

   ```bash
   fastapi dev server.py
    ```

3. Test with MCP client

   ```bash
   uv run client.py
    ```

## Tech Stack

- UV
- Python
- FastAPI
- MCP Protocol
